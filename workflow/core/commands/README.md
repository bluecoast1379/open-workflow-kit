# Core Commands

每个 command 文件都是一个阶段契约。工具可以把它暴露成 slash command、prompt、rule 或 checklist，但每个 adapter 都必须回指这些 core 文件。

阶段：

1. `/new-feature`
2. `/01-需求讨论`
3. `/02-产品文档`
4. `/03-技术架构`
5. `/03-06-研发准备`
6. `/04-代码实现`
7. `/04A-前端代码实现`
8. `/04B-后端代码实现`
9. `/05-代码审查`
10. `/06-测试用例`
11. `/07-测试执行`
12. `/08-验收表格`
13. `/09-验收`
14. `/10-培训文档`
15. `/11-上线邮件通知`
16. `/12-复盘总结`
17. `/workflow-status`

初始化器会把具体 command 文件写入目标工作区。
