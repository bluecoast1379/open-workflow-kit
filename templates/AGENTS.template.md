# Agent Workflow

This workspace uses the shared workflow generated by Agent Workflow Starter Kit.

## Source Of Truth

- Team profile: `workflow/team-profile.yaml`
- Core workflow: `workflow/core/`
- Feature deliverables: `features/{feature}/`
- Tool adapters: generated thin entries only

## Hard Gates

- Do not modify business code before the feature branch gate and implementation stage gate pass.
- Remote Git operations, branch creation, push, tag, merge, build/deploy triggers, database writes, and production config writes are manual-only.
- Same-repository parallel implementation requires separate worktrees after implementation stage starts.
- Missing local source materials must be requested from the user or recorded in `workflow/INITIALIZATION_QUESTIONS.md`.

## Tool Policy

Use the current tool's own adapter only. Do not promise identical behavior across all tools.

## Start

Read `workflow/team-profile.yaml`, then follow `workflow/core/commands/`.
